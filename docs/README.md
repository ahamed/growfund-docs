# Growfund Plugin Documentation

Welcome to the comprehensive documentation for the Growfund WordPress plugin. This documentation is designed to help administrators and site owners understand, install, configure, and use Growfund to build a professional fundraising platform.

## Overview

Growfund is a complete WordPress crowdfunding and donation plugin that enables you to collect funds directly from your website. Whether you're running a nonprofit organization, managing a charity, organizing community projects, or launching creative campaigns, Growfund provides everything you need to manage your fundraising activities.

The plugin offers two distinct operational modes:
- **Campaign Mode**: Reward-based crowdfunding with pledges and rewards
- **Donation Mode**: Direct donations with tribute functionality

## Key Features

- **Dual Fundraising Modes**: Choose between campaign-based crowdfunding or direct donations
- **Multi-Role System**: Separate dashboards for administrators, fundraisers, donors, and backers
- **Campaign Management**: Full-featured campaign creation with rich content and status management
- **Payment Gateways**: Support for Stripe, PayPal, offline payments, and WooCommerce integration
- **Reward System**: Create multiple reward tiers with different pledge amounts (Campaign Mode)
- **Email Notifications**: 40+ automated email templates to keep users informed
- **Analytics & Reporting**: Track campaign performance, donations, and key metrics
- **Customization Options**: Brand your platform with logos, colors, and custom styling

## Free vs Pro Features

Growfund is available in two versions:

**Free Version** includes:
- Campaign mode with up to 3 rewards (5 items per reward)
- Donation mode (basic features)
- Basic user management
- Payment gateway integration
- Basic email notifications
- Transaction tracking

**Pro Version** (Growfund Pro) adds:
- Unlimited rewards and reward items
- Campaign collaborators
- Campaign start date scheduling
- Post campaign updates
- Campaign overview analytics
- Donor overview dashboard
- Fundraiser management system
- Fund management system
- Anonymous contributions
- Advanced email notifications
- PDF receipt customization
- Branding customization
- Guest checkout
- Admin commission settings
- Comprehensive analytics
- And much more

Features marked with "(Pro)" throughout the documentation require Growfund Pro to be installed and activated.

## System Requirements

- **WordPress**: 5.9 or higher
- **PHP**: 7.4 or higher
- **MySQL**: 5.7+ or MariaDB 10.3+
- **Memory Limit**: 256MB minimum
