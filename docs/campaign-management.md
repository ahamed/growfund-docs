# Campaign Management

This guide explains how to create, manage, and configure campaigns in Growfund. Campaigns are the core of your fundraising platform, whether you're using Campaign Mode or Donation Mode.

## Overview

Campaigns are fundraising initiatives that allow you to collect funds from supporters. The process of creating and managing campaigns varies depending on whether you're using Campaign Mode or Donation Mode.

### Campaign Types

- **Crowdfunding Campaigns** (Campaign Mode): Reward-based campaigns with pledges and rewards
- **Donation Campaigns** (Donation Mode): Direct donation campaigns without rewards

## Creating a Campaign

### Prerequisites

Before creating a campaign:

1. **Verify Platform Mode**
   - Ensure you've selected the correct platform mode
   - Campaign Mode for reward-based campaigns
   - Donation Mode for direct donations

2. **Configure Settings**
   - Complete basic settings configuration
   - Set up payment gateways
   - Configure email notifications

3. **User Permissions**
   - Ensure you have permission to create campaigns
   - Administrators can always create campaigns
   - Fundraisers can create campaigns (Pro feature)

### Step-by-Step Campaign Creation

#### For Campaign Mode (Crowdfunding)

1. **Navigate to Campaigns**
   - Go to **Growfund** > **Campaigns**
   - Click **Create Campaign** button

2. **Basic Information**
   - Enter campaign title
   - Write campaign description
   - Upload featured image
   - Add campaign video (optional)
   - Set campaign category and tags

3. **Goal and Deadline**
   - Set funding goal amount
   - Set campaign deadline
   - Configure campaign duration
   - Set start date (Pro feature)

4. **Rewards** (Campaign Mode)
   - Add reward tiers (free: up to 3 rewards, 5 items per reward)
   - Set reward amounts and descriptions
   - Configure shipping options (if applicable)
   - Set reward delivery dates

5. **Additional Settings** (Pro features)
   - Add campaign collaborators
   - Configure campaign FAQs
   - Set suggested donation amounts
   - Allow custom donation amounts
   - Allow pledging without reward
   - Configure campaign visibility

6. **Submit Campaign**
   - Save as draft for later editing
   - Submit for admin approval (if required)
   - Publish immediately (if allowed)


#### For Donation Mode

1. **Navigate to Donations**
   - Go to **Growfund** > **Donations**
   - Click **Create Donation Campaign**

2. **Basic Information**
   - Enter campaign title
   - Write campaign description
   - Upload featured image
   - Add campaign video (optional)
   - Set campaign category and tags

3. **Donation Settings**
   - Set target amount (optional)
   - Configure donation form options
   - Set suggested donation amounts (Pro feature)
   - Allow custom donation amounts (Pro feature)

4. **Fund Assignment** (Pro feature)
   - Assign campaign to a fund
   - Organize campaigns by cause

5. **Tribute Options** (Pro feature)
   - Enable tribute donations
   - Configure tribute form fields
   - Set up tribute notifications

6. **Submit Campaign**
   - Save as draft for later editing
   - Submit for admin approval (if required)
   - Publish immediately (if allowed)


## Campaign Status

Campaigns can have different statuses throughout their lifecycle:

### Status Types

- **Draft**: Campaign is saved but not published
- **Pending**: Campaign is submitted for admin approval
- **Approved**: Campaign is approved and published
- **Declined**: Campaign is rejected by admin
- **Funded**: Campaign has reached its funding goal (Campaign Mode)
- **Ended**: Campaign deadline has passed
- **Active**: Campaign is currently accepting donations (Donation Mode)

### Status Workflow

1. **Draft**: Creator saves campaign as draft
2. **Pending**: Creator submits campaign for approval
3. **Approved**: Admin approves campaign
4. **Active/Funded**: Campaign is live and accepting contributions
5. **Ended**: Campaign deadline passes or is manually ended

## Managing Campaigns

### Viewing Campaigns

1. **Campaign List**
   - Go to **Growfund** > **Campaigns** (Campaign Mode)
   - Or **Growfund** > **Donations** (Donation Mode)
   - View all campaigns with filtering and search options

2. **Campaign Details**
   - Click on a campaign to view details
   - See campaign statistics and performance
   - View contributions and contributors


### Editing Campaigns

1. **Select Campaign**
   - Go to campaign list
   - Click **Edit** on the campaign you want to modify

2. **Update Campaign**
   - Modify campaign details
   - Update goal or deadline
   - Add or edit rewards (Campaign Mode)
   - Update campaign settings

3. **Save Changes**
   - Click **Update Campaign**
   - Changes are saved immediately

**Note**: Some changes may require admin approval depending on settings.

### Approving Campaigns (Administrator)

1. **Review Campaign**
   - Go to **Growfund** > **Campaigns**
   - Find campaigns with "Pending" status
   - Click to review campaign details

2. **Approve or Decline**
   - Click **Approve** to publish the campaign
   - Click **Decline** to reject with reason
   - Campaign creator receives notification


### Ending Campaigns

1. **Manual End**
   - Go to campaign details
   - Click **End Campaign**
   - Confirm campaign ending

2. **Automatic End**
   - Campaigns automatically end when deadline passes
   - Admin can configure automatic ending behavior

## Campaign Features

### Basic Features

- **Rich Content**: Text, images, videos, and media
- **Categories**: Organize campaigns by category
- **Tags**: Add tags for better discoverability
- **Search**: Advanced search and filtering
- **Bookmarking**: Users can bookmark favorite campaigns (Pro feature)

### Campaign Mode Features

- **Rewards**: Multiple reward tiers with different amounts
- **Pledges**: Supporters make pledges toward goals
- **Shipping**: Configure shipping for physical rewards
- **Reward Fulfillment**: Track reward delivery status

### Donation Mode Features

- **Direct Donations**: Simple donation process
- **Funds Organization**: Organize campaigns into funds (Pro feature)
- **Tribute Donations**: Donate in honor or memory (Pro feature)
- **Anonymous Donations**: Allow anonymous giving (Pro feature)

### Pro Features

- **Unlimited Rewards**: No limit on rewards or items (Pro)
- **Campaign Collaborators**: Multiple users can manage campaigns (Pro)
- **Campaign Start Date**: Schedule campaigns for future launch (Pro)
- **Post Updates**: Share campaign progress updates (Pro)
- **Campaign Copy**: Duplicate existing campaigns (Pro)
- **Campaign Overview**: Analytics and performance metrics (Pro)
- **Campaign FAQs**: Add frequently asked questions (Pro)
- **Suggested Amounts**: Pre-configured donation amounts (Pro)
- **Custom Donations**: Allow custom donation amounts (Pro)
- **Pledge Without Reward**: Support without selecting reward (Pro)
- **Continue After Goal**: Keep accepting contributions after goal (Pro)

## Best Practices

### Campaign Creation

- **Clear Goals**: Set realistic and specific funding goals
- **Compelling Content**: Write engaging descriptions with images
- **Regular Updates**: Keep supporters informed with updates (Pro)
- **Reward Planning**: Create attractive rewards (Campaign Mode)
- **Timeline**: Set appropriate deadlines

### Campaign Management

- **Regular Monitoring**: Check campaign performance regularly
- **Respond to Supporters**: Engage with contributors
- **Fulfill Rewards**: Deliver rewards promptly (Campaign Mode)
- **Update Campaigns**: Keep campaigns updated and active
