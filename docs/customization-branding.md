# Customization & Branding (Pro)

This guide explains how to customize and brand your Growfund platform with Pro features. Customization options allow you to match your platform's appearance to your organization's branding.

## Overview

Growfund Pro includes comprehensive customization and branding features that allow you to personalize your fundraising platform. These features help you create a cohesive brand experience for your donors and supporters.

**Note**: Customization and branding features are available in Growfund Pro. The free version includes basic customization options.

## Branding Settings

### Accessing Branding Settings

1. **Navigate to Branding**
   - Go to **Growfund** > **Settings** > **Branding**
   - View branding customization options
   - Configure brand colors and logo

2. **Branding Options**
   - Logo and branding
   - Color customization
   - Email template branding
   - PDF receipt branding

## Logo and Branding

### Adding Logo

1. **Navigate to Branding**
   - Go to **Growfund** > **Settings** > **Branding**
   - Find "Logo" section

2. **Upload Logo**
   - Click **Upload Logo** button
   - Select logo image file
   - Upload logo image

3. **Configure Logo**
   - Set logo size
   - Set logo position
   - Configure logo display

4. **Save Settings**
   - Click **Save Changes**
   - Logo is applied to platform

### Logo Display

Logo appears on:
- User dashboards
- Email templates
- PDF receipts
- Frontend pages (if configured)
