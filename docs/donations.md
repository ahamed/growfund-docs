# Donations

This guide explains how to create and manage donation campaigns in Donation Mode. Donation campaigns allow you to collect direct donations without offering rewards or perks.

## Overview

Donation Mode is designed for nonprofits, charities, and organizations that want to accept direct donations. Unlike Campaign Mode, donation campaigns don't involve rewards or pledgesâ€”supporters simply make donations to support your cause.

### Key Concepts

- **Donations**: Direct contributions from supporters
- **Donors**: Supporters who make donations
- **Donation Campaigns**: Campaigns that accept direct donations
- **Funds**: Organizational structure for grouping campaigns (Pro feature)
- **Tribute Donations**: Donations made in honor or memory of someone (Pro feature)

## Creating Donation Campaigns

### Prerequisites

Before creating donation campaigns:

1. **Verify Platform Mode**
   - Ensure Donation Mode is enabled
   - Go to **Growfund** > **Settings** > **General**
   - Verify "Donation Mode" is selected

2. **Configure Settings**
   - Complete basic settings configuration
   - Set up payment gateways
   - Configure donation form options

### Step-by-Step Donation Campaign Creation

1. **Navigate to Donations**
   - Go to **Growfund** > **Donations**
   - Click **Create Donation Campaign** button

2. **Basic Information**
   - Enter campaign title
   - Write campaign description
   - Upload featured image
   - Add campaign video (optional)
   - Set campaign category and tags

3. **Donation Settings**
   - Set target amount (optional)
   - Configure donation form options
   - Set default donation amounts
   - Configure suggested amounts (Pro feature)

4. **Fund Assignment** (Pro feature)
   - Assign campaign to a fund
   - Organize campaigns by cause or purpose
   - See [Funds](funds) for detailed information

5. **Tribute Options** (Pro feature)
   - Enable tribute donations
   - Configure tribute form fields
   - Set up tribute notifications
   - See [Tribute Donations](#tribute-donations-pro) section

6. **Additional Settings** (Pro features)
   - Enable anonymous donations
   - Allow company donations
   - Configure donation form options
   - Set campaign visibility

7. **Submit Campaign**
   - Save as draft for later editing
   - Submit for admin approval (if required)
   - Publish immediately (if allowed)

### Form Configuration

## Donation Processing

### Making Donations

1. **Frontend Donation**
   - Donor visits campaign page
   - Clicks donation button
   - Fills out donation form

2. **Select Amount**
   - Choose from suggested amounts
   - Enter custom amount (Pro feature)
   - Select donation type (one-time or recurring)

3. **Donor Information**
   - Enter name and email
   - Add billing address (if required)
   - Enter tribute information (Pro feature, if applicable)

4. **Payment Processing**
   - Select payment method
   - Enter payment details
   - Submit donation

5. **Confirmation**
   - Donor receives confirmation email
   - Receipt is generated automatically
   - Donation is recorded in system

### Donation Status

Donations can have different statuses:

- **Pending**: Donation is initiated but payment not yet processed
- **Completed**: Payment is successfully processed
- **Failed**: Payment processing failed
- **Refunded**: Donation is refunded to donor
- **Cancelled**: Donation is cancelled

## Managing Donations

### Viewing Donations

1. **Navigate to Donations**
   - Go to **Growfund** > **Donations** (transaction list)
   - View all donations across campaigns
   - Filter by campaign, status, or date

2. **Campaign Donations**
   - Go to campaign details
   - Click **Donations** tab
   - View donations for that specific campaign

### Donation Details

- **Donor Information**: Name, email, address
- **Donation Amount**: Amount donated
- **Payment Method**: Payment gateway used
- **Transaction ID**: Payment transaction identifier
- **Date and Time**: Donation timestamp
- **Receipt**: PDF receipt generation
- **Tribute Information**: Tribute details (Pro feature, if applicable)

### Processing Donations

1. **View Pending Donations**
   - Go to **Growfund** > **Donations**
   - Filter by "Pending" status
   - Review pending donations

2. **Process Payment**
   - Click on donation to view details
   - Verify payment information
   - Process payment through payment gateway

3. **Confirm Donation**
   - Mark donation as completed
   - Send confirmation email to donor
   - Generate receipt

## Anonymous Donations (Pro)

### Enabling Anonymous Donations

1. **Navigate to Settings**
   - Go to **Growfund** > **Settings** > **User Permissions**
   - Find "Anonymous Contributions" option

2. **Enable Feature**
   - Toggle anonymous donations on
   - Configure privacy settings
   - Set display options

3. **Save Settings**
   - Click **Save Changes**
   - Anonymous donations are now available

### Anonymous Donation Process

1. **Donor Selection**
   - Donor selects "Donate anonymously" option
   - Donor information is not displayed publicly
   - Donation still appears in campaign totals

2. **Privacy Settings**
   - Donor name is hidden from public display
   - Donor email is kept private
   - Admin can still view donor information

3. **Receipt Generation**
   - Receipt still generated for donor
   - Email sent to donor
   - Public display shows "Anonymous" instead of name

## Tribute Donations (Pro)

### Setting Up Tribute Donations

1. **Navigate to Campaign Settings**
   - Go to campaign settings
   - Find "Tribute" section
   - Enable tribute donations

2. **Configure Tribute Options**
   - Enable "In Honor" donations
   - Enable "In Memory" donations
   - Configure tribute form fields
   - Set up tribute notifications

3. **Save Settings**
   - Click **Save Changes**
   - Tribute donations are now available

### Tribute Donation Process

1. **Donor Selection**
   - Donor selects tribute option during donation
   - Chooses "In Honor" or "In Memory"
   - Enters honoree information

2. **Tribute Information**
   - Enter honoree name
   - Add tribute message (optional)
   - Enter notification recipient email (optional)

3. **Notification** (Pro feature)
   - Notification email sent to recipient
   - Includes tribute information
   - E-card option available (Pro feature)

### Managing Tribute Donations

- View tribute donations in campaign
- See tribute information and messages
- Manage tribute notifications
- Export tribute donation reports

## Best Practices

- **Clear Purpose**: Clearly state campaign purpose and goals

### Donation Processing

- **Quick Processing**: Process donations promptly
- **Clear Communication**: Keep donors informed
- **Receipt Delivery**: Ensure receipts are sent automatically
- **Thank You Messages**: Send personalized thank you messages

### Donor Management

- **Donor Engagement**: Engage with donors regularly
- **Appreciation**: Thank donors for their contributions
- **Updates**: Share campaign progress with donors
- **Privacy**: Respect donor privacy preferences
