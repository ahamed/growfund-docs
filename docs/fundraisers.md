# Fundraisers (Pro)

This guide explains how to manage fundraisers in Growfund Pro. The fundraiser management system allows you to assign fundraisers to campaigns, track their performance, and manage their activities.

## Overview

The fundraiser management system is a Pro feature that allows administrators to create, assign, and manage fundraisers. Fundraisers can create and manage their own campaigns, and administrators can track their performance and activity.

**Note**: Fundraiser management is available in Growfund Pro. The free version does not include fundraiser management features.

## Fundraiser Management

### Accessing Fundraisers

1. **Navigate to Fundraisers**
   - Go to **Growfund** > **Fundraisers**
   - View all fundraisers
   - Filter and search fundraisers

2. **Fundraiser List**
   - View all registered fundraisers
   - See fundraiser status and activity
   - View fundraiser performance metrics

### Creating Fundraisers

1. **Navigate to Fundraisers**
   - Go to **Growfund** > **Fundraisers**
   - Click **Add New Fundraiser** button

2. **Select User**
   - Choose existing user to promote to fundraiser
   - Or create new user and assign fundraiser role
   - Enter user information

3. **Configure Fundraiser**
   - Set fundraiser profile information
   - Assign to specific funds (if using funds)
   - Set fundraiser permissions
   - Configure fundraiser settings

4. **Save Fundraiser**
   - Click **Save Fundraiser**
   - Fundraiser can now create campaigns

### Fundraiser Profile

Fundraiser profile includes:

- **Personal Information**: Name, email, address
- **Fund Assignment**: Funds assigned to fundraiser
- **Campaigns**: Campaigns created by fundraiser
- **Performance**: Fundraiser performance metrics
- **Activity**: Fundraiser activity log

## Fundraiser Assignment

### Assigning Fundraisers to Campaigns

1. **Navigate to Campaign**
   - Go to **Growfund** > **Campaigns**
   - Select a campaign
   - Click **Edit**

2. **Assign Fundraiser**
   - Find "Fundraiser Assignment" section
   - Select fundraiser from dropdown
   - Or assign to fund (if using funds)

3. **Save Assignment**
   - Click **Update Campaign**
   - Fundraiser is assigned to campaign

### Assigning Fundraisers to Funds

1. **Navigate to Fund**
   - Go to **Growfund** > **Funds**
   - Select a fund
   - Click **Edit**

2. **Assign Fundraiser**
   - Find "Fundraiser Assignment" section
   - Select fundraiser from dropdown
   - Save assignment

3. **Save Fund**
   - Click **Update Fund**
   - Fundraiser is assigned to fund

## Fundraiser Dashboard

### Fundraiser Dashboard Features

Fundraisers have access to their own dashboard with:

- **My Campaigns**: Campaigns created by fundraiser
- **Campaign Analytics**: Performance metrics for campaigns
- **Pledges**: Pledge management (Campaign Mode)
- **Backers**: Backer management (Campaign Mode)
- **Donations**: Donation management (Donation Mode)
- **Donors**: Donor management (Donation Mode)
- **Profile**: Fundraiser profile management
- **Settings**: Fundraiser settings

### Fundraiser Profile Management

Fundraisers can manage their profiles:

1. **Navigate to Profile**
   - Go to fundraiser profile
   - Click **Edit Profile**

2. **Update Profile**
   - Update personal information
   - Upload profile picture
   - Update biographical information
   - Change password

3. **Save Profile**
   - Click **Save Profile**
   - Changes are saved

2. **View Performance Metrics**

   - Campaign success rate
   - Average campaign revenue

### Fundraiser Metrics

## Fundraiser Settings

### Configuring Fundraiser Settings

1. **Navigate to Settings**
   - Go to **Growfund** > **Settings** > **User Permissions**
   - Find "Fundraiser Settings" section

2. **Configure Settings**
   - Fundraiser permissions
   - Campaign creation permissions
   - Fund assignment permissions
   - Notification preferences

3. **Save Settings**
   - Click **Save Changes**
   - Settings are applied

### Fundraiser Permissions

Fundraiser permissions include:

- **Campaign Creation**: Can create campaigns
- **Campaign Editing**: Can edit own campaigns
- **Campaign Publishing**: Can publish campaigns directly
- **Fund Assignment**: Can be assigned to funds
- **Analytics Access**: Can view campaign analytics

## Fundraiser Management

### Editing Fundraisers

1. **Navigate to Fundraiser**
   - Go to **Growfund** > **Fundraisers**
   - Select a fundraiser
   - Click **Edit**

2. **Update Fundraiser**
   - Modify profile information
   - Change fund assignments
   - Update permissions
   - Configure settings

3. **Save Changes**
   - Click **Update Fundraiser**
   - Changes are saved

### Managing Fundraiser Status

Fundraiser status management:

1. **Navigate to Fundraiser**
   - Go to **Growfund** > **Fundraisers**
   - Select a fundraiser
   - View status options

2. **Change Status**
   - Activate fundraiser
   - Deactivate fundraiser
   - Suspend fundraiser

3. **Save Status**
   - Click **Update Status**
   - Status is updated

### Fundraiser Approval

Fundraiser approval process:

1. **Review Fundraiser**
   - Go to **Growfund** > **Fundraisers**
   - Find pending fundraisers
   - Review fundraiser application

2. **Approve or Decline**
   - Click **Approve** to approve fundraiser
   - Click **Decline** to reject with reason
   - Fundraiser receives notification

## Fundraiser Payouts

### Configuring Payouts

1. **Navigate to Fundraiser**
   - Go to **Growfund** > **Fundraisers**
   - Select a fundraiser
   - Click **Payouts** tab

2. **Configure Payouts**
   - Set payout schedule
   - Configure payout method
   - Set payout thresholds
   - Configure payout settings

3. **Save Settings**
   - Click **Save Changes**
   - Payouts are configured

### Processing Payouts

1. **View Pending Payouts**
   - Go to fundraiser payouts
   - View pending payouts
   - Review payout amounts

2. **Process Payout**
   - Select payout to process
   - Verify payout amount
   - Process payout through payment gateway

3. **Confirm Payout**
   - Mark payout as processed
   - Update payout status
   - Send confirmation to fundraiser

### Notification Settings

Configure fundraiser notifications:

1. **Navigate to Settings**
   - Go to **Growfund** > **Settings** > **Email & Notifications**

2. **Configure Notifications**

3. **Save Settings**
   - Click **Save Changes**
   - Notifications are configured

### Notification Types

Fundraiser notifications include:

- **Campaign Approved**: When campaign is approved
- **Campaign Declined**: When campaign is declined
- **Campaign Update**: When campaign update is posted
- **Performance Updates**: When performance milestones are reached
- **Payout Notifications**: When payouts are processed
