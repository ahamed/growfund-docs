# Funds (Pro)

This guide explains how to manage funds in Growfund Pro. The fund management system allows you to organize donation campaigns into funds for better organization and reporting.

## Overview

The fund management system is a Pro feature that allows you to create and manage funds to organize donation campaigns. Funds help you organize campaigns by cause, purpose, or category, making it easier to track performance and manage donations.

**Note**: Fund management is available in Growfund Pro. The free version does not include fund management features.

## Fund Management

### Accessing Funds

1. **Navigate to Funds**
   - Go to **Growfund** > **Funds**
   - View all funds
   - Filter and search funds

2. **Fund List**
   - View all created funds
   - See fund status and activity
   - View fund performance metrics

### Creating Funds

1. **Navigate to Funds**
   - Go to **Growfund** > **Funds**
   - Click **Create Fund** button

2. **Fund Information**
   - Enter fund name
   - Write fund description
   - Upload fund image (optional)
   - Set fund category (optional)

3. **Fund Settings**
   - Set fund visibility
   - Configure fund settings
   - Set fund target amount (optional)
   - Configure fund options

4. **Save Fund**
   - Click **Create Fund**
   - Fund is created and ready for campaigns

### Fund Profile

Fund profile includes:

- **Fund Information**: Name, description, image
- **Campaigns**: Campaigns assigned to fund
- **Donors**: Donors who contributed to fund
- **Performance**: Fund performance metrics
- **Analytics**: Fund analytics and reports

## Assigning Campaigns to Funds

### Assigning Campaigns

1. **Navigate to Campaign**
   - Go to **Growfund** > **Donations**
   - Select a campaign
   - Click **Edit**

2. **Assign to Fund**
   - Find "Fund Assignment" section
   - Select fund from dropdown
   - Or create new fund

3. **Save Assignment**
   - Click **Update Campaign**
   - Campaign is assigned to fund

### Fund Campaign Management

1. **Navigate to Fund**
   - Go to **Growfund** > **Funds**
   - Select a fund
   - Click **Campaigns** tab

2. **View Campaigns**
   - View all campaigns in fund
   - See campaign performance
   - Manage fund campaigns

3. **Add Campaigns**
   - Click **Add Campaign**
   - Select campaigns to add
   - Save assignment

## Fund Management

### Editing Funds

1. **Navigate to Fund**
   - Go to **Growfund** > **Funds**
   - Select a fund
   - Click **Edit**

2. **Update Fund**
   - Modify fund information
   - Update fund description
   - Change fund image
   - Update fund settings

3. **Save Changes**
   - Click **Update Fund**
   - Changes are saved

### Fund Settings

Configure fund settings:

1. **Navigate to Fund**
   - Go to **Growfund** > **Funds**
   - Select a fund
   - Click **Settings** tab

2. **Configure Settings**
   - Fund visibility
   - Fund target amount
   - Fund options
   - Fund permissions

3. **Save Settings**
   - Click **Save Changes**
   - Settings are applied

### Fund Visibility

Fund visibility options:

- **Public**: Fund is visible to all users
- **Private**: Fund is visible only to admins
- **Restricted**: Fund is visible to specific users

## Fund Reports

### Generating Fund Reports

1. **Navigate to Fund**
   - Go to **Growfund** > **Funds**
   - Select a fund
   - Click **Reports** tab

2. **Select Report Type**
   - Revenue reports
   - Campaign reports
   - Donor reports
   - Performance reports

3. **Configure Report**
   - Select date range
   - Choose metrics to include
   - Set report format

4. **Generate Report**
   - Click **Generate Report**
   - Report is generated
   - Download or email report

### Report Types

- **Revenue Reports**: Detailed revenue analysis
- **Campaign Reports**: Campaign performance reports
- **Donor Reports**: Donor analysis reports
- **Performance Reports**: Fund performance reports
