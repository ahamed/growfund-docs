# Security & Privacy

This guide explains the security and privacy features available in Growfund. These features help protect your fundraising platform, user data, and transactions.

## Overview

Growfund includes comprehensive security and privacy features designed to protect your platform, user data, and financial transactions. These features help ensure safe and secure fundraising operations.

## Security Features

### Email Verification (Pro)

Email verification helps ensure that only legitimate users can register accounts.

#### Enabling Email Verification

1. **Navigate to Security Settings**
   - Go to **Growfund** > **Settings** > **Security**
   - Find "Email Verification" option

2. **Enable Email Verification**
   - Toggle email verification on
   - Configure verification settings
   - Set verification requirements

3. **Save Settings**
   - Click **Save Changes**
   - Email verification is enabled

#### Email Verification Process

1. **User Registration**
   - User registers account
   - Verification email is sent
   - User clicks verification link

2. **Account Activation**
   - Account is activated after verification
   - User can now log in
   - Unverified accounts are restricted
