# Settings

This guide explains all the settings and configuration options available in Growfund. Settings allow you to customize your fundraising platform according to your needs.

## Overview

Growfund includes comprehensive settings that allow you to configure all aspects of your fundraising platform. These settings are organized into different categories for easy navigation and management.

## Accessing Settings

1. **Navigate to Settings**
   - Go to **Growfund** > **Settings**
   - View all setting categories
   - Configure platform settings

2. **Settings Categories**
   - General settings
   - Campaign settings
   - Payment settings
   - User permissions
   - Email notifications
   - PDF receipt
   - Security settings
   - Branding settings
   - Advanced settings


## General Settings

### Platform Mode

1. **Navigate to General Settings**
   - Go to **Growfund** > **Settings** > **General**
   - Find "Platform Mode" option

2. **Select Mode**
   - **Campaign Mode**: For reward-based crowdfunding
   - **Donation Mode**: For direct donations
   - Save selection

### Currency Settings

1. **Navigate to General Settings**
   - Go to **Growfund** > **Settings** > **General**
   - Find "Currency" section

2. **Configure Currency**
   - Select default currency
   - Set currency symbol
   - Configure currency format

3. **Save Settings**
   - Click **Save Changes**
   - Currency is applied

### Date and Time Settings

1. **Navigate to General Settings**
   - Go to **Growfund** > **Settings** > **General**
   - Find "Date Format" and "Time Zone" sections

2. **Configure Settings**
   - Select date format
   - Set time zone
   - Configure time format

3. **Save Settings**
   - Click **Save Changes**
   - Settings are applied
